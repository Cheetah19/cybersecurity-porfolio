# Cybersecurity Portfolio & Reconnaissance Reports

Welcome to my cybersecurity portfolio! This repository documents my journey through practical, portfolio-ready security projects and reconnaissance reports. 

## About This Repository
This space houses hands-on projects, technical breakdowns, and structured security write-ups completed as part of practical training and the **MyFirstHack** curriculum. The goal is to build real-world analytical skills, from foundational packet analysis to comprehensive external threat surface mapping.

## Ethical Standards & Methodology
All projects, assessments, and reconnaissance exercises contained within this repository strictly adhere to ethical, passive standards:
* **Passive Reconnaissance Only:** Utilizing open-source intelligence (OSINT) and publicly available records.
* **No Active Probing:** Zero intrusive scanning, exploitation, or unauthorized probing against target infrastructure.
* **Responsible Documentation:** Maintained with a focus on professional reporting standards and educational transparency.
